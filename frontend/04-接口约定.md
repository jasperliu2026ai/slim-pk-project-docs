## 一、基础约定

### 请求格式

- 协议：HTTPS（开发阶段允许 HTTP + 不校验合法域名）
- 基础 URL：`https://api.slimpk.com/api`（生产）；`http://<CVM-IP>:3000/api`（开发联调）
- 数据格式：`Content-Type: application/json`
- 鉴权：`Authorization: Bearer <JWT>`

### 响应格式（统一）

```json
{
  "code": 0,
  "message": "ok",
  "data": {}
}
```

### 错误码

| code | 含义 | 前端处理 |
|------|------|----------|
| 0 | 成功 | 正常处理 |
| 1001 | 未登录 / token 无效 | 跳登录页 |
| 1002 | token 过期 | 静默重新登录后重试 |
| 2001 | 参数错误 | toast 错误 message |
| 2002 | 今日已打卡（体重） | 前端处理打卡状态 |
| 2003 | 已加入该 PK | 直接跳 ranking 页 |
| 2004 | PK 未开始 | toast 提示 |
| 2005 | PK 已结束 | toast + 跳结算页 |
| 2006 | 体重超出合理范围 | toast 提示 |
| 5000 | 服务端错误 | toast「服务异常，请稍后重试」 |

## 三、前端 request 封装（utils/request.ts）

```typescript
interface RequestOptions {
  url: string;
  method?: 'GET' | 'POST' | 'PUT' | 'DELETE';
  data?: Record<string, unknown>;
  showLoading?: boolean;
}

async function request<T>(options: RequestOptions): Promise<T> {
  const { url, method = 'GET', data, showLoading = false } = options;

  if (showLoading) wx.showLoading({ title: '加载中', mask: true });

  const token = wx.getStorageSync('token');

  return new Promise((resolve, reject) => {
    wx.request({
      url: BASE_URL + url,
      method,
      data,
      header: {
        'Content-Type': 'application/json',
        Authorization: token ? `Bearer ${token}` : '',
      },
      success: (res) => {
        const body = res.data as { code: number; message: string; data: T };

        if (body.code === 1001 || body.code === 1002) {
          // token 无效 → 重新登录
          reLogin();
          return;
        }

        if (body.code !== 0) {
          wx.showToast({ title: body.message || '请求失败', icon: 'none' });
          reject(new Error(body.message));
          return;
        }

        resolve(body.data);
      },
      fail: () => {
        wx.showToast({ title: '网络异常，请重试', icon: 'none' });
        reject(new Error('network error'));
      },
      complete: () => {
        if (showLoading) wx.hideLoading();
      },
    });
  });
}
```
