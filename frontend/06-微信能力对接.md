## 一、微信登录

### 完整时序

```
前端                              后端
 │                                 │
 │── wx.login() ──────────────────▶│ 获取 code
 │◀── code ───────────────────────│
 │                                 │
 │── POST /api/auth/login ─────────▶│
 │   { code, nickName, avatarUrl } │
 │                                 │── wx.auth.code2Session(code)
 │                                 │   → { openid, session_key }
 │                                 │── 生成 JWT（payload: userId, openid）
 │◀── { token, userId, isNewUser } ─│
 │                                 │
 │ isNewUser === true              │
 │── 弹「敏感信息单独同意弹框」        │
 │── 用户点击「同意」              │
 │── wx.reLaunch 到首页            │
```

### 代码实现

```typescript
// pages/login/index.ts
async function handleLogin() {
  try {
    wx.showLoading({ title: '登录中', mask: true });

    // Step 1: 获取微信 code
    const { code } = await new Promise<WechatMiniprogram.LoginSuccessCallbackResult>(
      (resolve, reject) => wx.login({ success: resolve, fail: reject })
    );

    // Step 2: 获取用户昵称和头像
    // 注意：wx.getUserProfile 已废弃，改用 open-type="getUserInfo" 的 button
    // 昵称和头像在用户点击按钮时通过 e.detail 获取
    const { nickName, avatarUrl } = userInfoFromButton;

    // Step 3: 后端换 token
    const result = await request<LoginResponse>({
      url: '/auth/login',
      method: 'POST',
      data: { code, nickName, avatarUrl },
    });

    userStore.setUserInfo(result);

    // Step 4: 首次登录弹隐私弹框
    if (result.isNewUser) {
      await showPrivacyDialog();
    }

    wx.reLaunch({ url: '/pages/index/index' });
  } catch (err) {
    wx.showToast({ title: '登录失败，请重试', icon: 'none' });
  } finally {
    wx.hideLoading();
  }
}
```

### 注意事项

- `wx.getUserProfile` 在基础库 2.27.1 后已废弃，获取用户信息必须使用 `<button open-type="getUserInfo">`
- 头像和昵称的获取改为：`<button open-type="chooseAvatar">` + `<input type="nickname">`（基础库 2.21.2+）
- token 有效期：由后端决定（建议 7 天），过期后前端静默重新走 `wx.login` 流程

## 三、订阅消息（打卡提醒）

### 流程

1. 用户首次进入打卡页时，调用 `wx.requestSubscribeMessage`
2. 用户同意后，后端保存订阅状态
3. 每日 20:00（或用户自定义时间），后端调用微信服务器发送订阅消息

### 前端代码

```typescript
// 订阅打卡提醒
async function subscribeCheckinReminder(pkId: string) {
  try {
    const { CHECKIN_TEMPLATE_ID } = await getConfig(); // 从后端获取模板 ID

    const result = await new Promise<WechatMiniprogram.RequestSubscribeMessageSuccessCallbackResult>(
      (resolve, reject) =>
        wx.requestSubscribeMessage({
          tmplIds: [CHECKIN_TEMPLATE_ID],
          success: resolve,
          fail: reject,
        })
    );

    if (result[CHECKIN_TEMPLATE_ID] === 'accept') {
      // 通知后端用户已同意订阅
      await request({ url: '/subscription/confirm', method: 'POST', data: { pkId } });
    }
  } catch (err) {
    // 订阅失败不影响打卡，静默忽略
    console.warn('订阅消息失败', err);
  }
}
```

### 模板内容（需在微信公众平台配置）

- 模板类别：运动健康 / 健身提醒
- 关键词：时间（2026-07-14 20:00）+ 提醒内容（「你今日还未打卡，加油！」）+ 跳转路径

## 四、图片上传（打卡照片）

```typescript
// utils/upload.ts
export async function uploadCheckinPhoto(filePath: string, pkId: string): Promise<string> {
  return new Promise((resolve, reject) => {
    wx.uploadFile({
      url: `${BASE_URL}/api/upload`,
      filePath,
      name: 'file',
      formData: { pkId },
      header: {
        Authorization: `Bearer ${wx.getStorageSync('token')}`,
      },
      success: (res) => {
        const body = JSON.parse(res.data) as { code: number; data: { fileUrl: string } };
        if (body.code === 0) {
          resolve(body.data.fileUrl);
        } else {
          reject(new Error('上传失败'));
        }
      },
      fail: () => reject(new Error('网络异常')),
    });
  });
}
```

## 五、保存图片到相册

```typescript
// 保存海报到相册（需要 scope.writePhotosAlbum 权限）
async function saveToAlbum(tempFilePath: string) {
  try {
    // 先检查权限
    const { authSetting } = await wx.getSetting();

    if (!authSetting['scope.writePhotosAlbum']) {
      await wx.authorize({ scope: 'scope.writePhotosAlbum' });
    }

    await wx.saveImageToPhotosAlbum({ filePath: tempFilePath });
    wx.showToast({ title: '已保存到相册', icon: 'success' });
  } catch (err) {
    if ((err as { errMsg: string }).errMsg.includes('auth deny')) {
      // 引导用户去设置页开权限
      wx.showModal({
        title: '需要相册权限',
        content: '请在设置中开启「保存到相册」权限',
        confirmText: '去设置',
        success: (res) => {
          if (res.confirm) wx.openSetting();
        },
      });
    }
  }
}
```

## 六、小程序码生成

小程序码由**后端生成**（需要 `access_token`，前端无法直接调用），流程：

```
前端请求 GET /api/pk/{pkId}/qrcode
    ↓
后端调用微信 API: wx.getUnlimitedQRCode
    ↓
返回 base64 编码的图片数据
    ↓
前端解码 → 显示 / 绘制到 Canvas
```
