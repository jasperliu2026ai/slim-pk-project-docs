## 一、状态管理选型

**选型：MobX-miniprogram + miniprogram-mobx-bindings**

| 方案 | 优缺点 | 结论 |
|------|--------|------|
| 页面 data 自治 | 简单，但跨页面同步麻烦（如打卡后首页需要刷新） | 局部场景用 |
| 手写全局 store（globalData） | 无响应式，需手动通知更新 | 不用 |
| **MobX-miniprogram** | 响应式 + 声明式绑定；官方推荐方案 | ✅ 选用 |
| Redux | 模板代码多，小程序无官方绑定 | 不用 |

## 二、Store 设计

### 2.1 `user.store.ts` — 用户信息

```typescript
import { observable, action, computed } from 'mobx-miniprogram';

export const userStore = observable({
  // State
  userId: '' as string,
  openId: '' as string,
  nickname: '' as string,
  avatarUrl: '' as string,
  accessToken: '' as string,
  tokenExpireAt: 0 as number,
  privacyAccepted: false as boolean,

  // Computed
  get isLoggedIn(): boolean {
    return !!this.accessToken && Date.now() < this.tokenExpireAt;
  },

  get displayName(): string {
    return this.nickname || '减肥小达人';
  },

  // Actions
  login: action(function(this: typeof userStore, data: LoginResponse) {
    this.userId = data.userId;
    this.openId = data.openId;
    this.nickname = data.nickname;
    this.avatarUrl = data.avatarUrl;
    this.accessToken = data.accessToken;
    this.tokenExpireAt = data.expireAt;
    // 持久化
    wx.setStorageSync('accessToken', data.accessToken);
    wx.setStorageSync('tokenExpireAt', data.expireAt);
  }),

  logout: action(function(this: typeof userStore) {
    this.userId = '';
    this.accessToken = '';
    this.tokenExpireAt = 0;
    wx.removeStorageSync('accessToken');
    wx.removeStorageSync('tokenExpireAt');
  }),

  acceptPrivacy: action(function(this: typeof userStore) {
    this.privacyAccepted = true;
    wx.setStorageSync('privacyAccepted', true);
  }),
});
```

### 2.2 `pk.store.ts` — PK 状态

```typescript
import { observable, action, computed } from 'mobx-miniprogram';
import type { PKItem, PKDetail } from '../types/pk';

export const pkStore = observable({
  // State
  pkList: [] as PKItem[],
  pkListLoading: false as boolean,
  pkListPage: 1 as number,
  pkListHasMore: true as boolean,

  currentPk: null as PKDetail | null,
  currentPkLoading: false as boolean,

  todayCheckedIn: false as boolean,

  // Computed
  get ongoingPks(): PKItem[] {
    return this.pkList.filter(pk => pk.status === 'ongoing');
  },

  get myActivePkCount(): number {
    return this.ongoingPks.length;
  },

  // Actions
  setPkList: action(function(this: typeof pkStore, list: PKItem[], append = false) {
    if (append) {
      this.pkList = [...this.pkList, ...list];
    } else {
      this.pkList = list;
    }
  }),

  setCurrentPk: action(function(this: typeof pkStore, pk: PKDetail | null) {
    this.currentPk = pk;
    if (pk) {
      this.todayCheckedIn = pk.myCheckinToday ?? false;
    }
  }),

  markCheckedIn: action(function(this: typeof pkStore) {
    this.todayCheckedIn = true;
    if (this.currentPk) {
      this.currentPk.myCheckinToday = true;
    }
    // 同步更新列表中的打卡状态
    const idx = this.pkList.findIndex(p => p.id === this.currentPk?.id);
    if (idx !== -1) {
      this.pkList[idx] = { ...this.pkList[idx], myCheckinToday: true };
    }
  }),

  reset: action(function(this: typeof pkStore) {
    this.pkList = [];
    this.pkListPage = 1;
    this.pkListHasMore = true;
    this.currentPk = null;
    this.todayCheckedIn = false;
  }),
});
```

## 三、页面中使用 Store

使用 `miniprogram-mobx-bindings` 的 `ComponentWithStore` / `createStoreBindings`：

```typescript
// pages/index/index.ts
import { ComponentWithStore } from 'miniprogram-mobx-bindings';
import { pkStore, userStore } from '../../store/index';

ComponentWithStore({
  storeBindings: [
    {
      store: pkStore,
      fields: ['pkList', 'pkListLoading', 'pkListHasMore'],
      actions: ['setPkList'],
    },
    {
      store: userStore,
      fields: ['displayName', 'avatarUrl'],
    },
  ],

  data: {
    activeTab: 'ongoing',
  },

  methods: {
    async loadPkList() {
      pkStore.pkListLoading = true;
      try {
        const res = await getPkList({ page: pkStore.pkListPage });
        pkStore.setPkList(res.data.list, pkStore.pkListPage > 1);
        pkStore.pkListHasMore = res.data.hasMore;
        pkStore.pkListPage++;
      } finally {
        pkStore.pkListLoading = false;
      }
    },
  },
});
```

## 四、持久化策略

| 数据 | 存储方式 | 说明 |
|------|----------|------|
| accessToken | `wx.setStorageSync` | 登录态，启动时恢复 |
| tokenExpireAt | `wx.setStorageSync` | 过期时间戳 |
| privacyAccepted | `wx.setStorageSync` | 隐私政策已同意标记 |
| pkList / pkDetail | 不持久化 | 每次进页面重新拉取，保证数据新鲜 |
| 打卡图片 | 不本地缓存 | 上传后立即释放临时文件 |

**启动恢复逻辑（app.ts）**
```typescript
App({
  onLaunch() {
    // 恢复登录态
    const token = wx.getStorageSync('accessToken');
    const expireAt = wx.getStorageSync('tokenExpireAt');
    if (token && Date.now() < expireAt) {
      userStore.accessToken = token;
      userStore.tokenExpireAt = expireAt;
    }
    // 恢复隐私同意状态
    const privacyAccepted = wx.getStorageSync('privacyAccepted');
    if (privacyAccepted) {
      userStore.privacyAccepted = true;
    }
  }
});
```

## 五、状态流转图（打卡核心链路）

```
用户进入 pk-detail
       │
       ▼
  pkStore.currentPk.myCheckinToday?
       │
    ┌──┴──┐
   是      否
    │       │
显示「今日   显示「去打卡」按钮
已打卡」态        │
                  ▼
            跳转 checkin 页
                  │
         填写体重 + 拍照水印
                  │
            提交打卡 API
                  │
           成功 → pkStore.markCheckedIn()
                  │
              ← navigateBack 到 pk-detail
                  │
          pk-detail onShow 刷新
          （因 pkStore.todayCheckedIn = true
            显示已打卡状态）
```
