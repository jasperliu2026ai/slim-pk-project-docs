## 一、WeightInput（体重输入组件）

### 职责

取代系统数字键盘，提供统一的体重输入体验，包含范围校验和提交锁定。

### 组件接口

```typescript
// components/weight-input/index.ts
Component({
  properties: {
    // 已有体重（编辑模式时传入）
    defaultValue: { type: Number, value: 0 },
    // 是否锁定（今日已打卡时禁止输入）
    disabled: { type: Boolean, value: false },
    // 单位标签
    unit: { type: String, value: 'kg' },
  },
  methods: {
    // 用户点击确认后触发
    // detail: { value: number }
    onConfirm(e: WechatMiniprogram.CustomEvent) {},
    onCancel() {},
  }
})
```

### 关键逻辑

```typescript
const MIN_WEIGHT = 30.0;
const MAX_WEIGHT = 300.0;

function validate(input: string): { valid: boolean; msg?: string } {
  const v = parseFloat(input);
  if (isNaN(v)) return { valid: false, msg: '请输入数字' };
  if (v < MIN_WEIGHT || v > MAX_WEIGHT) {
    return { valid: false, msg: `请输入合理体重（${MIN_WEIGHT}-${MAX_WEIGHT}kg）` };
  }
  // 保留1位小数
  return { valid: true };
}
```

### 设计说明

- 自定义数字键盘（0-9 + 小数点 + 退格），不调用系统键盘（系统键盘在某些 Android 上遮挡严重）
- 提交后进入 `disabled` 状态，不允许当日再次修改（初始体重用）
- 每日打卡体重是否允许覆盖：**待 DR-006 确认**，默认按「允许覆盖，取最后一次」实现

## 二、CheckinCamera（拍照打卡组件）

### 职责

强制调用摄像头（禁止相册），拍照后在 Canvas 2D 叠加时间水印，上传到后端。

### 完整流程

```
用户点击「拍照」
    │
    ▼
wx.chooseMedia({ sourceType: ['camera'], mediaType: ['image'] })
    │ 获得临时图片路径 tempFilePath
    ▼
wx.compressImage({ src: tempFilePath, quality: 80 })  // 压缩到 < 300KB
    │ 得到压缩后路径
    ▼
Canvas 2D 合成水印
    ├── 绘制原图
    ├── 叠加文字：日期时间（2026-07-14 16:17:45）
    └── 叠加文字：用户昵称（脱敏：前2字+**）
    │
    ▼
canvas.toTempFilePath()  // 得到水印图临时路径
    │
    ▼
wx.uploadFile({
  url: 'http://<后端地址>/api/upload',
  filePath: watermarkedPath,
  name: 'file',
  header: { Authorization: 'Bearer <token>' }
})
    │ 返回 { fileUrl: 'https://cos.../xxx.jpg' }
    ▼
触发 onUploadSuccess 事件，将 fileUrl 传给父页面
```

### 组件接口

```typescript
Component({
  properties: {
    userId: { type: String, required: true },
    nickname: { type: String, required: true },
  },
  methods: {
    // 上传成功后触发，detail: { fileUrl: string }
    onUploadSuccess() {},
    // 上传失败后触发，detail: { error: string }
    onUploadFail() {},
  }
})
```

### 水印绘制代码（Canvas 2D）

```typescript
async function drawWatermark(
  ctx: WechatMiniprogram.CanvasContext,
  imagePath: string,
  width: number,
  height: number,
  nickname: string
): Promise<void> {
  // 绘制原图
  ctx.drawImage(imagePath, 0, 0, width, height);

  // 水印背景（半透明黑条）
  ctx.fillStyle = 'rgba(0, 0, 0, 0.45)';
  ctx.fillRect(0, height - 60, width, 60);

  // 时间文字
  const now = new Date();
  const timeStr = formatDateTime(now); // 2026-07-14 16:17:45
  ctx.fillStyle = '#FFFFFF';
  ctx.font = 'bold 22px sans-serif';
  ctx.fillText(timeStr, 16, height - 32);

  // 昵称（脱敏）
  const maskedName = maskNickname(nickname); // 前2字 + **
  ctx.font = '18px sans-serif';
  ctx.fillStyle = 'rgba(255,255,255,0.8)';
  ctx.fillText(maskedName, 16, height - 10);
}

function maskNickname(name: string): string {
  if (name.length <= 2) return name + '**';
  return name.slice(0, 2) + '**';
}
```

### 安全说明

> ⚠️ 前端水印可被绕过（截图替换）。L1 熟人圈阶段依赖信任；L2/L3 可升级为后端校验水印哈希。

### 降级方案

- 如果 Canvas 合成在低端 Android 上 OOM → 降级压缩分辨率到 720P
- 超时 5 秒 → 提示「合成失败，请重试」，保留原图路径（不上传无水印图）

## 三、ScoreRankItem（排行榜条目组件）

### 展示内容

```
┌─────────────────────────────────────────┐
│  🥇  [头像]  昵称（前2字）         88.5分│
│      减重 -5.2%   连续打卡 12天         │
└─────────────────────────────────────────┘
```

### 组件接口

```typescript
interface RankItem {
  rank: number;              // 排名（1, 2, 3...）
  avatarUrl: string;         // 头像 URL
  nickname: string;          // 昵称（已脱敏）
  totalScore: number;        // 综合积分（保留1位小数）
  weightLossPct: number;     // 减重百分比（如 -5.2 表示减了 5.2%）
  checkinDays: number;       // 连续打卡天数
  isSelf: boolean;           // 是否当前用户（高亮处理）
}

Component({
  properties: {
    item: { type: Object as () => RankItem, required: true },
  }
})
```

### 隐私规则

- **不展示绝对体重数字**（初始/当前体重均隐藏）
- 只展示减重百分比（如 -5.2%）
- 头像正常展示；昵称展示前 2 字 + **（一致脱敏）
- 当前用户条目：橙色背景高亮，始终可见（即使排名在屏幕外也展示在底部固定条）

## 四、PosterCanvas（战报海报组件）

### 海报尺寸

- 输出分辨率：750 × 1334 px（3:5.34 比例，适合微信朋友圈 / 私信）
- Canvas 实际大小：375 × 667 rpx（Retina 下 2x 导出）

### 绘制层次

```
Layer 1: 渐变背景（橙 #FF6B2B → 深橙 #E55A1F）
Layer 2: Logo 图片 + 「减肥 PK」文字（左上）
Layer 3: PK 名称（大字，居中）
Layer 4: 周期文字（小字，居中）
Layer 5: 分隔线
Layer 6: 个人数据区
    ├── 起始体重（大号数字 + kg）
    ├── → 箭头
    ├── 结束体重（大号数字 + kg）
    ├── 减重量（-X.Xkg）
    ├── 减重%（红色大字，如 -5.2%）
    └── 名次（🥇 第1名）
Layer 7: 小程序码（右下角，通过后端接口生成 base64 图）
Layer 8: 底部文案「扫码加入我的下一场 PK 💪」
```

### 关键 API

```typescript
// 生成海报并保存
async function generatePoster(data: PosterData): Promise<string> {
  const canvas = await getOffscreenCanvas(750, 1334);
  const ctx = canvas.getContext('2d');

  await drawBackground(ctx, 750, 1334);
  await drawLogo(ctx, logoPath);
  await drawPKInfo(ctx, data.pkName, data.period);
  await drawPersonalData(ctx, data);
  await drawQRCode(ctx, data.qrBase64);

  return new Promise((resolve, reject) => {
    canvas.toTempFilePath({
      x: 0, y: 0,
      width: 750, height: 1334,
      destWidth: 750, destHeight: 1334,
      fileType: 'jpg', quality: 0.92,
      success: (res) => resolve(res.tempFilePath),
      fail: reject,
    });
  });
}
```
