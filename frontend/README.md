## 🚀 快速上手

### 环境要求

- **微信开发者工具**：≥ 1.06.2405020
- **Node.js**：≥ 18.x
- **npm**：≥ 9.x

### 本地启动

```bash
# 1. 克隆前端小程序代码
git clone https://github.com/jasperliu2026ai/slim-pk-miniapp.git
cd slim-pk-miniapp

# 2. 安装依赖
npm install

# 3. 微信开发者工具打开项目根目录
# 4. 工具 → 构建 npm
# 5. 编译运行
```

### 分支规范

| 分支 | 用途 |
|------|------|
| `main` | 线上稳定版本 |
| `dev` | 开发集成分支 |
| `feat/xxx` | 功能开发 |
| `fix/xxx` | Bug修复 |
| `hotfix/xxx` | 紧急线上修复 |
