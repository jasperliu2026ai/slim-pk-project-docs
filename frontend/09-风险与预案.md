## 四、回滚方案

### 功能开关（前端）

通过后端接口返回的 `featureFlags` 字段控制功能可用性，不上线前关闭：

```typescript
interface FeatureFlags {
  enablePosterCanvas: boolean;    // 战报海报（Canvas 版）
  enableLottieAnimation: boolean; // lottie 动效
  enableSubscribeMessage: boolean; // 订阅消息
}

// 关掉战报 Canvas → 降级到文字版结算页
// 关掉 lottie → 降级到静态图标
```

### 版本回滚

- 如果正式版发现严重 Bug：在微信公众平台「撤回此次发布」→ 回退到上一版
- 撤回后立即通知 v_juvnliu，说明原因 + 修复时间

### 数据安全

- 前端不存储体重原始数据，回滚不涉及数据恢复问题
- 打卡照片已上传 COS 的数据不会因前端回滚丢失
