## 快速索引

### 云函数清单

| 函数名 | 触发方式 | 职责 |
|--------|----------|------|
| `auth-login` | 主动调用 | 微信登录、创建/更新用户 |
| `user-update` | 主动调用 | 更新昵称/头像 |
| `user-delete` | 主动调用 | 删除账号（合规要求） |
| `room-create` | 主动调用 | 创建 PK 房间 |
| `room-join` | 主动调用 | 加入房间、录入初始体重 |
| `room-get` | 主动调用 | 获取房间详情 |
| `room-list-mine` | 主动调用 | 我参与的 PK 列表 |
| `checkin-submit` | 主动调用 | 提交打卡（体重/饮食/运动） |
| `checkin-today` | 主动调用 | 今日打卡状态查询 |
| `leaderboard-get` | 主动调用 | 排行榜（读快照） |
| `score-calc` | 定时 00:05 | T+1 积分快照计算 |
| `pk-settle` | 定时 00:10 + 主动 | PK 结算、生成战报数据 |

### 积分公式速查

```
综合积分 = 减重百分比×K×70% + 体重打卡率×15% + 饮食完成率×8% + 运动完成率×7%
K = 1（减重10%对应70分；K值待产品拍板后更新）
```

### 错误码速查

| 错误码 | 含义 |
|--------|------|
| 0 | 成功 |
| 1001 | 参数校验失败 |
| 1002 | 未登录 |
| 1003 | 无权限 |
| 2001 | 房间不存在 |
| 2002 | 已在房间中 |
| 2003 | PK 已结束 |
| 3001 | 今日已打卡（幂等场景不报错） |
| 3003 | 体重超出合理范围（30-300kg） |
| 3004 | 体重变化异常（与初始偏差>40%或单日>5kg） |
| 5000 | 系统内部错误 |
