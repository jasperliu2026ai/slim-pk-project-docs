## 5. SSL 证书配置（Let's Encrypt）

```bash
# 安装 acme.sh
curl https://get.acme.sh | sh

# 申请通配符证书（DNS-01 验证，需腾讯云 DNSPod API）
export TENCENTCLOUD_SECRET_ID="xxx"
export TENCENTCLOUD_SECRET_KEY="xxx"

acme.sh --issue \
  -d yourdomain.com \
  -d "*.yourdomain.com" \
  --dns dns_tencent \
  --keylength ec-256

# 安装到 Nginx
acme.sh --install-cert -d yourdomain.com \
  --cert-file /etc/nginx/ssl/cert.pem \
  --key-file /etc/nginx/ssl/key.pem \
  --fullchain-file /etc/nginx/ssl/fullchain.pem \
  --reloadcmd "nginx -s reload"

# 开启自动续期
acme.sh --install-cronjob
```

## 6. 微信小程序合法域名配置

微信公众平台 → 开发 → 开发管理 → 开发设置 → 服务器域名：

| 类型 | 配置值 | 用途 |
|------|--------|------|
| request 合法域名 | `https://api.yourdomain.com` | API 接口调用 |
| uploadFile 合法域名 | `https://api.yourdomain.com` | 打卡照片上传 |
| downloadFile 合法域名 | `https://cdn.yourdomain.com` | 下载素材/图片 |

**配置要求**：
- 必须 HTTPS（备案 + 有效证书）
- 不能是 IP 地址，不能带端口号
- 每月最多修改 5 次

## 7. DNS 配置

```
yourdomain.com.      A     CLB 公网 IP
api.yourdomain.com.  A     CLB 公网 IP
cdn.yourdomain.com.  CNAME xxxx.cdn.dnsv1.com（COS CDN 地址）
```

> 腾讯云 DNS 解析生效：通常 5-10 分钟，全球最长 72 小时。
