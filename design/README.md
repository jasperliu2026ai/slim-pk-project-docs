# 减肥 PK 小程序｜设计交付包 v1.0

角色：UU【设计】
范围：微信小程序 MVP，包括用户登录、创建和加入 PK、每日打卡、排行榜、结算海报、隐私合规底盘。
设计方向：精密仪器感 + 温暖陪伴感，克制高科技视觉，不贩卖身材焦虑。

## 关键设计原则

- 默认保护隐私：排行榜和分享卡不展示绝对体重，仅展示相对进度、积分、连续打卡。
- 鼓励式反馈：失败和掉队使用中性表达，不使用羞辱、恐吓、焦虑文案。
- 数据可信：体重、积分、排行等数字使用等宽数字，单位和精度明确。
- 组件可复用：PK 卡片、打卡卡片、进度环、排行榜行、结算海报沉淀为组件。
