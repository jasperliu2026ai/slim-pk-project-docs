# 视觉规范 v1.0

## 1. 风格关键词

精密：数据像仪器一样清晰可信。
温暖：反馈陪伴式，不冰冷、不羞辱。
克制科技感：深空底、单一亮色、干净网格、精准数字。

## 2. 色彩

| Token | 色值 | 用途 |
|---|---|---|
| color-bg-deep | #0A0E1A | 深色主背景 |
| color-bg-page | #F6F8FB | 浅色页面背景 |
| color-surface | #FFFFFF | 卡片背景 |
| color-surface-dark | #111827 | 深色卡片 |
| color-primary | #32D583 | 成长绿，成功、进度、主行动 |
| color-accent | #FF8A3D | 活力橙，高光、提醒、结算 |
| color-text-primary | #101828 | 主文本 |
| color-text-secondary | #667085 | 次文本 |
| color-border | #EAECF0 | 分割线 |
| color-warning-soft | #FFF4E5 | 温和提醒背景 |

一票否决：不使用荧光紫粉赛博配色、不满屏玻璃拟态、不做发光 3D 图表。

## 3. 字体

中文正文：苹方或 HarmonyOS Sans，14 到 16px，字重 400 到 500。
页面标题：苹方或 HarmonyOS Sans，20 到 24px，字重 600。
大数字：DIN、SF Mono 或 JetBrains Mono，32 到 56px，字重 700。
图表刻度：SF Mono 或 JetBrains Mono，11 到 12px，字重 500。

## 4. 间距与圆角

space-4 为 4px，space-8 为 8px，space-12 为 12px，space-16 为 16px，space-24 为 24px。
radius-sm 为 8px，radius-md 为 12px，radius-lg 为 16px，radius-xl 为 24px。

## 5. 组件规范

PKCard：展示 PK 名称、剩余天数、我的排名、今日打卡状态、主 CTA。状态包括进行中、未开始、已结束、已解散。

CheckinInput：体重输入使用大数字 + kg 单位，精度默认 0.1kg；短期变化异常时展示温和提醒，不用红色报错。

LeaderboardRow：左侧排名，中间头像昵称，右侧积分或进度。Top3 使用轻量勋章，当前用户行使用浅绿背景强调。

PrivacySheet：用于照片、分享卡、数据展示前的二次确认。默认最小可见范围。

## 6. 动效

页面切换 220ms ease-out；卡片出现 280ms spring；数字变化 300ms ease-out；进度环 400ms spring；成功反馈 320ms spring + haptic。

## 7. 无障碍

正文对比度不低于 WCAG AA 4.5:1。点击热区不小于 44×44px。状态不只靠颜色区分，必须有文字或图标。体重、照片等敏感模块提供清晰撤回和可见范围入口。
