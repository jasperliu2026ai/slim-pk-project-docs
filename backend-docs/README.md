## 项目关键信息

| 项 | 值 |
|----|---|
| 上线目标 | 2026-08-01（L1 体验版）|
| 技术选型 | 微信云开发 CloudBase（DR-002 决策）|
| 运行时 | Node.js 16+ / TypeScript strict |
| 数据库 | 云数据库（文档型，MongoDB-like）|
| 文件存储 | 腾讯云 COS（通过云开发接入）|
| 定时任务 | 云函数定时触发器（每日 00:05 积分结算）|
| 开发环境数量 | 3 个（dev / staging / prod）|
| 云函数数量 | 12 个 |
| 数据库集合数量 | 6 个 |
